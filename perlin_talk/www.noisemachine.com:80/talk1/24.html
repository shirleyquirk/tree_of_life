Using 3D noise to animate 2D turbulent flow
  • Flame

Next I'm going to show some examples of how to animate flowing textures using noise. The important point to come away with here is that:
You need to use 3D noise in order to do convincing time-varying flow of 2D surfaces.
That extra dimension is needed because turbulent flow has a beautiful randomly coherent appearance that is best handled by treating time as another spatial (and therefore texturable) dimension.
