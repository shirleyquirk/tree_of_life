Ken Perlin's pure Java 1.0 interactive 3D renderer gallery

A lot of people are trying to convince you that you can't do real-time 3D rendering on the web wser without using some company's proprietary plug-in. I knew that was bogus, so as a home project I wrote a 3D renderer that will run in any web browser supporting Java 1.0. Below are some examples. - Ken

  • Animated character: My Best Fiend, the Politician (source code is here)
  • Example of multiple layers of transparency (source code is here)
  • Artifact from a Happier World (source code is here)
  • Sculpture: Linking Arms (over 100K triangles) (source code is here)

Feel free to use this software for academic or research (ie: non-commercial) projects. The base source code is in the following files:

RenderApplet.java         The applet that the above examples all extend.
Renderer.java This is what actually does the work of rendering.
Shape.java Toolkit to represent and build various shapes.
Noise.java My noise function (eg: used in the Sculpture example).
Matrix.java Support for 4x4 matrix transforms.
Vec.java Support for 3D vector operations.

I'm going to continue to improve the renderer, and to add cool new examples and capabilities, so watch this space.
